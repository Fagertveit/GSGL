GameSaw Game Library

GameSaw Game Library is a HTML5 game engine that makes prototyping 2d games dead easy using the HTML5 canvas element.

The GSGL consists of some common game engine components that is used when making games, such as basic shapes for collision detection, basic image manipulation and resource handling

This page will have a collection of GSGL apps that will show it's main components and give a basic instruction on how to use these.
